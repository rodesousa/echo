# Je Eerste Project Aanmaken

Leer hoe je je eerste Dembrane project opzet en configureert voor succes.

## Wat Je Gaat Leren
- Een nieuw Dembrane project aanmaken en een naam geven via het dashboard
- Projectcontext en documentatie toevoegen
- Transcripties en projectinstellingen beheren
- Voorbereiden op portalconfiguratie


## Stap 1: Je dashboard instellen

1. Log in op je Dembrane account
2. Klik op het **Instellingen** icoon rechtsboven in je dashboard om de taal van je dashboard te wijzigen of een probleem te melden. Als je op enig moment wilt uitloggen, kan dat hier ook.

![](/img13.png)

## Stap 2: Project Aanmaken

1. Dembrane maakt je eerste project voor je! Als je een nieuw project wilt maken, klik dan op het Home icoon en klik vervolgens op "Aanmaken".

![](/img1.png)

### Project Instellingen

Nu kom je in je hoofddashboard terecht. Zorg eerst dat je in het tabblad **"Project Instellingen"** bent. Hier moet je een aantal zaken instellen:

#### Project Bewerken

1. **Naam**: Geef je project een naam in plaats van **"Nieuw Project"**. Kies iets beschrijvends en herkenbaars. Je kunt dit later nog wijzigen
2. **Context**: Voeg details toe over het doel van je project (intern gebruik). Dit kan bijvoorbeeld zijn wat het doel van het project is

    Later kun je al je transcripties exporteren op deze pagina door op <span style={{color: 'blue'}}>Alle Transcripties Downloaden</span> te klikken

Als je je hele project wilt verwijderen, klik dan op <span style={{color: 'red'}}>Project Verwijderen</span>

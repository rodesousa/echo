# Voorbeeldvragen Gids

## Overzicht
Leer van praktische voorbeelden hoe je effectieve vragen voor analyse kunt formuleren.

## Voorbeeldvragen
### Voor Overzichtsanalyse
- "Wat zijn de belangrijkste besproken onderwerpen?"
- "Toon me patronen in klantfeedback"
- "Identificeer veelvoorkomende pijnpunten"

### Voor Gedetailleerde Analyse
- "Vergelijk reactietijden tussen teams"
- "Analyseer sentimenttrends over tijd"
- "Haal specifieke productfeedback eruit"

## Vraagsjablonen
1. Patroonanalyse
   ```
   Welke patronen komen naar voren in [onderwerp] tijdens [tijdsperiode]?
   ```
2. Vergelijkende Analyse
   ```
   Hoe verhoudt [metric A] zich tot [metric B] in termen van [criteria]?
   ```
3. Trendanalyse
   ```
   Welke trends kun je identificeren in [gebied] tussen [datum] en [datum]?
   ```

## Tips voor Betere Vragen
- Wees specifiek over tijdsperiodes
- Definieer duidelijke metrieken
- Gebruik consistente terminologie
- Verdeel complexe vragen in delen

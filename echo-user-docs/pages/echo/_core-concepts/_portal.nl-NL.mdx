# Deelnemersportal
De deelnemersportal is ECHO's hoofdinterface voor deelnemers. Het is een webapplicatie die iedereen in staat stelt om hun stem toe te voegen aan het gesprek.

## Hoe gebruik je de Deelnemersportal

1. Maak een project aan in het [Dembrane Dashboard](https://dashboard.dembrane.com)
2. Scan de QR-code of klik op de link om het gesprek te starten.
3. Voeg je stem toe aan het gesprek
4. Deel de link met anderen om hen aan het gesprek te laten deelnemen!

## Functies
De deelnemersportal heeft een aantal geweldige functies:
- Het is stem-eerst, dus iedereen kan zijn gedachten uitspreken zonder te typen. Maak je geen zorgen, er is ook een tekstinvoer voor toegankelijkheid.
- Het is instellen en vergeten, zodat je je kunt concentreren op het gesprek.
- Je kunt het anoniem gebruiken, of om namen vragen.
- Je kunt het op elk apparaat gebruiken, en het is toegankelijk vanaf elke browser.
- Je kunt om tags vragen, zodat je de gegevens later kunt filteren.
- Je kunt veel verschillende talen instellen en automatisch vertalen, zodat je iedereen kunt horen!
- Je kunt het tutorialniveau aanpassen om de ervaring af te stemmen op deelnemers die het nog nooit hebben gebruikt.
- Je kunt elke inhoud toevoegen aan de opnamepagina, om het aantrekkelijker te maken of feedback te verzamelen over specifieke inhoud.

> Zo simpel is het!

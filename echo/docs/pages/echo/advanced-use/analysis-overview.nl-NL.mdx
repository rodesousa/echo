# Analyse Overzicht
In deze tutorialreeks leer je hoe je het dashboard optimaal kunt gebruiken. We behandelen verschillende belangrijke gebieden om je te helpen efficiënt door grote hoeveelheden gespreksdata te navigeren.

## Wat Je Gaat Leren
- Effectieve chatstrategieën voor gedetailleerde analyse
- Grote datasets beheren met de Bibliotheek
- Gevorderde Prompt Engineering
- Je eigen audiobestanden uploaden

## Analyse Workflow Overzicht
1. Begin met de Bibliotheek om brede thema's in de hele dataset te identificeren
2. Verdeel de dataset in kleinere groepen met behulp van tags
3. Gebruik Chat voor gedetailleerde analyse van de kleinere groepen (minder dan 10 uur aan gesprekken)
4. <span style={{'color': 'grey'}}>Exporteer inzichten waar nodig _(binnenkort beschikbaar)_</span>

## Technische Beperkingen
- Maximaal aanbevolen gesprekken in chat: ~10 uur totaal
- De bibliotheek heeft geen groottelimieten, maar het zal langer duren om te maken en nieuwe perspectieven te genereren als je een grote dataset hebt.

# Kernconcepten

De kernconcepten van ECHO zijn:

- **Bibliotheek** - De bibliotheek is de plek waar je inzichten kunt creëren uit AL je gesprekken.
- **Views** - Views zijn perspectieven waarmee je alle gesprekken in een project kunt analyseren.
- **Insights** - Insights zijn clusters van citaten die ECHO heeft gecreëerd uit alle gesprekken in een project.
- **Stel vraag** - De Stel vraag interface is waar je kunt chatten met een selectie van gesprekken.
- **Portal** - De portal is waar deelnemers kunnen deelnemen aan het gesprek.
